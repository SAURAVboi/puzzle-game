# Puzzle Game

A fun and crazy jigsaw puzzle game with advanced features and smooth gameplay.

## Features
- Irregular puzzle pieces  
- Piece rotation (multiple angles)  
- Custom image selection  
- Mouse & touchscreen support  
- 12 to 50 of pieces 🧩 
- Zoom & pan  
- Save / load progress  

## Tech
HTML, CSS, JavaScript
